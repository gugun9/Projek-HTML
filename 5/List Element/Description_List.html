istilah istilah dalam pemrograman

Program
Kumpulan instruksi ini kemudian dieksekusi untuk melakukan sebuah tugas atau fungsi tertentu. Untuk memproses sebuah program, dibutuhkan CPU (central processing unit) komputer.
Agile
pendekatan software development yang mempercepat proses kerja. Dengan metode ini, kamu akan bekerja dalam porsi yang kecil-kecil tetapi mudah diselesaikan. Hasilnya, proyek yang dikerjakan bisa berkualitas tinggi, dan biaya yang dikeluarkan pun minimal.
API
menyampaikan perintah dari pengguna pada sistem. Dengan perintah ini, program bisa mengetahui respons seperti apa yang harus diberikan.
Algoritma
langkah-langkah atau instruksi atau urutan untuk menyelesaikan suatu masalah pemrograman komputer.
Bug
masalah yang terjadi dalam kode pemrograman.
